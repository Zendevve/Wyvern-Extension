# Changelog

All notable changes to Wyvern Drive will be documented in this file.

## [1.2.0] - 2024-12-15

### Added
- **Photo Timeline** - Google Photos-like gallery view with automatic date grouping
- **Offline Support** - Access your file list even without internet connection
- **File sharing** - Generate shareable links for individual files
- **PWA support** - Install Wyvern Drive as a standalone app

### Improved
- Faster file loading with optimized download queue
- Better thumbnail generation for images and videos
- Enhanced mobile responsiveness

### Fixed
- Extension connection stability improvements
- Blob URL cleanup issues resolved

---

## [1.1.0] - 2024-12-01

### Added
- **End-to-end encryption** - Secure your files with AES-256-GCM encryption
- **Multiple webhooks** - Spread storage across multiple Discord channels
- **File versioning** - Keep track of file history

### Improved
- Upload progress indicators
- File grid view with thumbnails

---

## [1.0.0] - 2024-11-15

### Initial Release
- Upload and download files via Discord webhooks
- Folder organization
- File management (rename, move, delete)
- Basic search functionality
